便携部署说明

1. 本地启动
   把整个文件夹复制到任意 Windows 电脑后，双击根目录的“启动网站.bat”。
   启动成功后打开：
   http://127.0.0.1:18080/

2. 开机自启和巡检
   进入 server 文件夹，右键管理员运行：
   安装开机自启和巡检.bat

   该脚本会用当前文件夹路径注册计划任务。移动文件夹后需要重新运行一次。

3. 公网访问
   本地服务器不依赖 Cloudflare，可以直接运行。
   如果需要继续使用 pzq1688.com 这种公网域名，需要在新电脑上安装 cloudflared，并准备可用的 tunnel 凭据和配置。

   推荐做法：
   - 安装 cloudflared
   - 在 server 文件夹放置 cloudflared-config.local.yml
   - 确保配置里的 credentials-file 指向新电脑上真实存在的凭据文件

   如果没有 cloudflared 或没有配置，启动脚本会自动只启动本地服务器，不会阻止网站运行。

4. 数据位置
   上传文件、ESP32 状态、账号、日志都保存在当前文件夹下的 dynamic 目录。
   复制整个文件夹时，这些数据会一起带走。
